---
title: Multi-modal Observations from Long- and short-read Alignments
description: Modeling single-nucleotide mismatches in all kinds of RNA-seq data
---

`mola` is a Python CLI for linking single-nucleotide mismatches back to individual RNA-seq reads, then carrying that read-level context into downstream counting, classification, and optional phasing.

It is aimed at bioinformaticians working with:

- bulk short-read RNA-seq
- single-cell short-read RNA-seq with barcodes and UMIs
- bulk long-read RNA-seq
- single-cell long-read RNA-seq
