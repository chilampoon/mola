`mola` is organized into four command groups:

- `mola read`
- `mola mut`
- `mola infer`
- `mola parse`

## Command families

### `mola read`

Read- and alignment-level preprocessing:

- `demux`
- `subset`
- `trim`
- `annotate`

### `mola mut`

Mismatch-to-read mapping and site-table export:

- `map`
- `write`

### `mola infer`

Optional probabilistic downstream steps:

- `posteriors`
- `phayes`
- `soma`

### `mola parse`

Count-table and matrix export:

- `count`
